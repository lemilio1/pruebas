# Proyecto ONVIA – Pruebas con Codex

## Battle Quiz Royale

Este repositorio contiene un prototipo de navegador para **Battle Quiz Royale**. Es un juego de disparos en primera persona inspirado en FreeFire y Fortnite, ambientado en una versión futurista de la Segunda Guerra Mundial. Incluye zombis como enemigos y un sencillo auto tipo *Lamborghini*. Para desbloquear armas se deben responder preguntas de cultura general.

### Ejecutar

Abre el archivo `game/index.html` en tu navegador. Al presionar **Iniciar** se activan los controles estilo FPS y aparece el punto de mira. Con **WASD** te mueves, clic izquierdo dispara y con **Q** contestas preguntas para conseguir armas. Acércate al auto y presiona **E** para conducirlo.

Esta versión es aún básica pero ya cuenta con:

- Movimiento en primera persona y disparos.
- Zombis que se acercan al jugador y reaparecen al ser eliminados.
- Preguntas de cultura general para desbloquear armas.
- Un auto sencillo que puede conducirse con las mismas teclas.

Falta mucho para llegar a un juego comercial completo, pero sirve como base para seguir agregando niveles, modelos detallados y compatibilidad móvil.
