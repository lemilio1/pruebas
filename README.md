# Proyecto ONVIA – Pruebas con Codex

## Battle Quiz Royale

Este repositorio contiene un prototipo muy simple de un juego en primera persona que se ejecuta directamente en el navegador. La idea es mezclar elementos de FreeFire y Fortnite en una ambientación futurista de la Segunda Guerra Mundial. El usuario puede responder preguntas de cultura general para desbloquear nuevas armas.

### Ejecutar

Abre el archivo `game/index.html` en tu navegador. Al presionar el botón **Iniciar** se activará el control de vista en primera persona (requiere usar el ratón). Presiona la tecla **Q** para responder preguntas y ganar armas imaginarias.

Esta versión es solo un esqueleto inicial: muestra el escenario 3D básico con un "zombi" y está lista para extenderse con más funcionalidades como vehículos, múltiples niveles y soporte móvil.
